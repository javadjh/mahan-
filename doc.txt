 سامانه مدیریت بایگانی ماهان
 فاز اول:
 این نرم افزار بر پایه وب طراحی و توسعه یافته و به منظور دیجیتال کردن اسناد و مدارک یک سازمان مورد استفاده قرار میگیره
 سرور:
 سرور این سامانه با فرم ورک Node.js کد نویسی شده و از معماری mvc برای مدیریت بهتر کد ها بهره بردیم . همچنین command ها و query های هر بخش جدا شده
  که باعث دسترسی راحت تر به اجزا مختلف پروژه می شود. همچنین دیتابیس این سامانه بر پایه no-sql طراحی شده و دیتابیس mongodb برای این موضوع انتخاب شده است به دلیل محبوبیت و قدرت بالای این دیتابیس
  در این پروژه جهت سهولت مدیریت دیتابیس از odm استفاده شده که mongoose نام دارد

  -authUtility:
  برای چک کردن دسترسی ها استفاده شده و همچنین از jwt برای ساخت توکن استفاده شده است(فایل .env دارای کلید jwt میباشد)
  الگوریتم:
  ابتدا id کاربر که در توکن وجود دارد دریافت میکنیم سپس آن را در دستابیس سرچ میکنیم سپس دسترسی های آن یوزر را دریافت میکنیم و چک میکنیم آیا به آن بایگانی و آن بخش دسترسی دارد یا خیر
  (لیست دسترسی های سامانه در handle/initProject وجود دارد)
  در پوشه آپلود هم فایل های مربوط به اسناد ذخیره می شود
